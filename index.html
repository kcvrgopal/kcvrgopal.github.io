SightMyDroid
  •  April 2014 to Present
SightMyDroid is a very simple and easy to use tool, which assists a user by providing his/her device's location remotely through sms. User can set some code of his choice in the application. Once the code has been set, All the user has to do to get his device's location is to simply text the code he/she has set initially to his device from any mobile phone using sms. SightMyDroid replies back to the user with the device's location using sms.
Working of the Application:
* User is prompted to create a PIN on the first launch of the application. Once the PIN is set, he/she is prompted to set an access code of his choice which serves as the key for finding his device.
* Once the PIN and Code are set, User can send a sms to his/her device with the code.
* The User gets back a sms from his device with the device's location
Technologies: Android, SQLite
